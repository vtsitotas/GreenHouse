## How a unit is built (current, working — NM-native)

1. **Pi Imager:** Raspberry Pi OS Lite **32-bit** (Pi Zero W = ARMv6). Edit Settings: hostname `greenhouse`, SSH on, user `pi`/`greenhouse2026`, WiFi *with internet* + country `GR`, locale `Europe/Athens`.
2. **Install from PC:**
   ```powershell
   $pi = "C:\Users\billy\Desktop\diplomatikh\pi"
   ssh pi@greenhouse.local "mkdir -p /home/pi/greenhouse"
   scp -r "$pi\scripts" "$pi\systemd" "$pi\portal" "$pi\mosquitto" "$pi\install.sh" pi@greenhouse.local:/home/pi/greenhouse/
   ssh pi@greenhouse.local 'find /home/pi/greenhouse -type f \( -name "*.sh" -o -name "*.py" -o -name "*.service" -o -name "*.conf" \) -exec sed -i "s/\r$//" {} +; sudo bash /home/pi/greenhouse/install.sh && sudo bash /home/pi/greenhouse/scripts/selftest.sh'
   ```
   Expect `RESULT: 16 passed, 0 failed`. (install.sh randomizes the pi password → `/boot/firmware/INITIAL_PASSWORD.txt`; admin SSH key keeps you in.)
3. **Make the golden image (once):** `ssh pi@greenhouse.local "sudo systemd-run --collect --unit=prep bash /home/pi/greenhouse/scripts/prep_image.sh"` → powers off → pull SD → read to `greenhouse.img` with Win32DiskImager.
4. **Mass-produce:** flash `greenhouse.img` to any SD → boots ready; each unit self-generates its own TLS certs, MQTT password, OS password, and `Greenhouse-XXXX` SSID on first boot.

> NOTE: `git clone` from the GitHub repo can replace the scp+CRLF step on the Pi once the repo is the source of truth — cleaner. (CRLF strip kept for now; `.gitattributes` is a pending nicety.)

## Key files (pi/)

| File | Role |
|---|---|
| `pi/install.sh` | **Master installer** (idempotent): packages, TLS, Mosquitto, services, admin key, captive DNS |
| `pi/scripts/gen_certs.sh` | Generates this unit's unique TLS CA+server cert (idempotent) |
| `pi/scripts/first_boot.sh` | Per-unit identity: TLS certs, MQTT pw + device.json, random OS pw |
| `pi/scripts/ap_up.sh` | AP via `nmcli` shared mode (open, 192.168.4.1) + captive 80→8080 redirect |
| `pi/scripts/selftest.sh` | Verify a provisioned unit (16 checks) |
| `pi/scripts/prep_image.sh` | Strip per-unit identity + dev WiFi, power off (run detached) |
| `pi/systemd/greenhouse-{firstboot,portal,ap}.service` | systemd units |
| `pi/portal/portal.py` | Flask :8080 — `/` setup form, `/connect` + `/api/connect` (nmcli), `/pair` JSON |
| `pi/mosquitto/mosquitto.conf` | conf.d fragment: `per_listener_settings`, 1883/8883/9001 |
